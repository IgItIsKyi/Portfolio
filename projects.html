12 Stocks Auto

This is a Python script that automatically buys and sells stocks based on the 12 stock strategy. It uses the Alpaca API to execute trades and can be run on a schedule using a task scheduler or cron job. The script is designed to be easy to use and will continue to be updated with new features as I progress in the idea. It is a great tool for anyone looking to automate their stock trading process and start building a portfolio with a diverse set of options.
